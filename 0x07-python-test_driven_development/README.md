# 0X07 PYTHON - TEST-DRIVEN DEVELOPMENT

This directory contains programs that have a specific function and a directory called 'tests' that will contain test files of each program, in order to learn Test-driven development

## 0-add_integer.py, tests/0-add_integer.txt

Function that adds 2 integers and its respective test file.

## 2-matrix_divided.py, tests/2-matrix_divided.txt

Function that divides all elements of a matrix, and its respective test file.

## 3-say_my_name.py, tests/3-say_my_name.txt

Function that prints 'My name is <first name> <last name>', and its respective test file.

## 4-print_square.py, tests/4-print_square.txt

Function that prints a square with the character '#' and its respective test file.

## 5-text_indentation.py, tests/5-text_indentation.txt

Function that prints a text with 2 new lines after each of these characters: ., ? and :, and its respective test file.

## tests/6-max_integer_test.py

Unittests for the function max_integer.
